``mongosh`` is available in different Linux installation packages:

.. list-table::
   :header-rows: 1

   * - Package Type
     - Linux Platform

   * - ``.deb``
     - - Ubuntu
       - Debian

   * - ``.rpm``
     - - RHEL (Red Hat Enterprise Linux)
       - Amazon Linux 2

   * - ``.tgz``
     - Not platform specific

Select the instruction tab that corresponds to the package you want to
use.
