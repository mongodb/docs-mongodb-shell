========================
Write Custom Log Entries
========================

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

You can write custom :ref:`MongoDB Shell log entries <mdb-shell-logs>`
from your :ref:`MongoDB Shell scripts <mdb-shell-write-scripts>`. Custom
log entries help with debugging, error handling, and alert you when the
script performs specific functions.

About this Task
---------------

MongoDB Shell supports the following log levels for custom log entries:

- ``log.debug()``
- ``log.error()``
- ``log.fatal()``
- ``log.info()``
- ``log.warn()``

Steps
-----

.. procedure::

   .. step:: Create a script that writes a custom log entry

      The following script inserts documents into the ``movies``
      collection and writes a custom ``info`` log entry. If the script
      errors, it writes a custom ``error`` log entry instead.

      .. code-block:: javascript
         :emphasize-lines: 22,24

         // connect-and-insert-with-log-entry.js
         
         try {
            db = connect( 'mongodb://localhost/myDatabase' );

            db.movies.insertMany( [
               {
                  title: 'Titanic',
                  year: 1997,
                  genres: [ 'Drama', 'Romance' ]
               },
               {
                  title: 'Spirited Away',
                  year: 2001,
                  genres: ['Animation', 'Adventure', 'Family' ]
               },
               {
                  title: 'Casablanca',
                  genres: [ 'Drama', 'Romance', 'War' ]
               }
            ] )
            log.info('InsertData: [$(new Date().toISOString()}] Inserted 3 movies')
         } catch (error) {
            log.error('ERROR: [$(new Date().toISOString()}] Insert failed. Reason: ${error.message}');
         }

      Save the script as ``connect-and-insert-with-log-entry.js``.

   .. step:: Run the script

      To run the ``connect-and-insert-with-log-entry.js`` script, use
      ``mongosh`` to connect to your deployment and run the following
      command within MongoDB Shell:

      .. code-block:: javascript

         load("connect-and-insert-with-log-entry.js")

Results
-------

The custom log entry appears in the logs for your session:

.. code-block:: javascript
   :copyable: false

   {"t":{"$date":"2025-02-25T15:41:49.135Z"},"s":"I","c":"MONGOSH-SCRIPTS","id":1000000054,"ctx":"custom-log","msg":"InsertData: 2025-02-25T15:41:49.135Z Inserted 3 movies"}
   
For more information about log sessions and how to retrieve log
messages, see :ref:`mdb-shell-view-logs`.
